<p align="center">
  <h1>Oisin UI</h1>
</p>

Oisin UI is a self-hosted web terminal for AI coding agents.

## Repo layout

- `packages/server` — Backend daemon and API surface
- `packages/cli` — Command-line client for managing agents

## Quick start

```bash
npm install
npm run dev:server
```

Then use the CLI to connect and control agents.
