**Left column — file list:**

State: `selectedFile: ParsedDiffFile | null` (useState, default null).
Auto-select: `useEffect` — when `unstagedFiles` or `stagedFiles` change, if `selectedFile` is no longer in either list, reset to null. Also auto-select first file if none selected and files exist: pick first from `unstagedFiles`, else first from `stagedFiles`.

Sections:
- "Unstaged" section: shown only if `unstagedFiles.length > 0`
- "Staged" section: shown only if `stagedFiles.length > 0`
- If both empty and not loading: show `<p className="p-3 text-sm text-muted-foreground">No changes</p>`

Each section uses `Collapsible` (default open). Section header (CollapsibleTrigger):
```tsx
<CollapsibleTrigger asChild>
  <Button type="button" variant="ghost" className="w-full justify-between px-3 py-1.5 text-xs font-semibold uppercase tracking-wide text-muted-foreground">
    <span>Unstaged ({unstagedFiles.length})</span>
    <ChevronDown className={cn("h-3 w-3 transition-transform", open && "rotate-180")} />
  </Button>
</CollapsibleTrigger>
```

Section body (CollapsibleContent): map over files, render `<DiffFileRow>` for each:
```tsx
<DiffFileRow
  key={file.path}
  file={file}
  selected={selectedFile?.path === file.path}
  onSelect={setSelectedFile}
/>
```

Files sorted alphabetically within each section (sort by `file.path` in the component, don't mutate props).

**Right column — diff viewer:**

If `selectedFile === null`:
```tsx
<p className="p-4 text-sm text-muted-foreground">Select a file to view its diff</p>
```

If `selectedFile` is binary or too_large:
```tsx
<p className="p-4 text-xs text-muted-foreground">
  {selectedFile.status === 'binary' ? 'Binary file. Diff preview unavailable.' : 'File exceeds diff preview limit.'}
</p>
```

- Use `useEffect` to compute html when `selectedFile` changes: `setHtml(toDiff2Html(selectedFile, { hunkLimit: selectedFile.hunks.length }))`
- Render: `<div className="diff2html-wrapper p-2" dangerouslySetInnerHTML={{ __html: html }} />`
- Show `<Skeleton className="m-4 h-32" />` while html is null (loading state)
