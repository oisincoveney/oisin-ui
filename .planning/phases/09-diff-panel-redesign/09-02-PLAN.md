**Rendering rules (all ShadCN/Tailwind, no raw HTML except allowed `<form>`):**

Use `Button` (variant="ghost") as the row trigger — full width, left-aligned, `type="button"`.

Layout inside the button:
- Left: filename display (see path logic below) + full path in Tooltip
- Right: +/- counts + status badge

**Path display logic** (reuse `getDiffFileDisplayPath` from diff2html-adapter):
- For renamed files (`oldPath !== path`): show `old → new` (getDiffFileDisplayPath already handles this)
- Truncate long paths from the START: show the last ~40 chars with `…` prefix if longer. Use CSS `direction: rtl; text-overflow: ellipsis; overflow: hidden; white-space: nowrap` trick via Tailwind: `truncate` class on a container with `dir="rtl"` and `text-align: left` override. Actually simpler: use a `<span>` with `className="block truncate text-sm font-medium"` and let CSS truncate the end — the full path is in the Tooltip so truncation direction doesn't matter much. Keep it simple: just `truncate` (truncates end).
- Wrap the path text in a `Tooltip` (TooltipProvider + Tooltip + TooltipTrigger + TooltipContent) showing the full `getDiffFileDisplayPath(file)`.

**+/- counts:**
```tsx
<span className="text-xs text-emerald-500">+{file.additions}</span>
<span className="text-xs text-rose-500 ml-1">-{file.deletions}</span>
```

**Status badges** (use Tailwind inline spans, not ShadCN Badge — Badge not in AGENTS.md list):
- `file.isNew`: small dot or text label `new` in muted green: `<span className="ml-1 text-[10px] font-medium text-emerald-400 uppercase tracking-wide">new</span>`
- `file.isDeleted`: `<span className="ml-1 text-[10px] font-medium text-rose-400 uppercase tracking-wide">del</span>`
- `file.status === 'binary'`: `<span className="ml-1 text-[10px] text-muted-foreground uppercase tracking-wide">bin</span>`
- `file.status === 'too_large'`: `<span className="ml-1 text-[10px] text-muted-foreground uppercase tracking-wide">large</span>`

**Selected state:** When `selected`, apply `bg-muted/60` background to the button (use `cn()` to conditionally add it).
